# React 计算器

这是一个使用 React 和 TypeScript 构建的现代化计算器应用。

## 功能特点

- 基本的算术运算（加、减、乘、除）
- 清除功能
- 实时显示计算过程
- 现代化的用户界面
- 响应式设计

## 安装

1. 克隆项目到本地：
```bash
git clone [repository-url]
```

2. 安装依赖：
```bash
npm install
```

3. 启动开发服务器：
```bash
npm start
```

## 使用方法

- 点击数字按钮输入数字
- 点击运算符按钮（+、-、×、÷）进行计算
- 点击等号（=）查看结果
- 点击 C 按钮清除当前输入

## 技术栈

- React 18
- TypeScript
- CSS Grid 布局
- React Hooks

## 开发环境要求

- Node.js 14.0.0 或更高版本
- npm 6.0.0 或更高版本
